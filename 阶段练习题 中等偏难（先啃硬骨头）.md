## 🎯 C++ STL 练习题推荐（巩固 vector / queue / list / set / map）

------

### 题1 — 约瑟夫问题

**📍 洛谷 P1996 · 难度：入门** **🔗** `https://www.luogu.com.cn/problem/P1996` **容器：`queue`**

n 个人围成一圈，每数到第 m 个人就出圈，输出出圈顺序。经典 `queue` 模拟题，用 `push` / `pop` / `front` 循环操作，不需要复杂算法，非常适合上手练习队列。

------

### 题2 — 木材仓库

**📍 洛谷 P5250 · 难度：普及−** **🔗** `https://www.luogu.com.cn/problem/P5250` **容器：`set`**

维护一个木材集合，支持插入和查询与给定值最接近的元素。需要用 `set` 的 `lower_bound` / `upper_bound` 来快速查找相邻元素，很好地锻炼有序集合的查询能力。

------

### 题3 — 保龄球

**📍 洛谷 P1918 · 难度：普及−** **🔗** `https://www.luogu.com.cn/problem/P1918` **容器：`map`**

给出每个球瓶编号对应的位置，查询哪些位置的球瓶互为"配对"。用 `map<int,int>` 建立编号到位置的映射，逻辑简单，是理解 map 键值对查询的好入门题。

------

### 题4 — 两数之和

**📍 力扣 LeetCode 1 · 难度：简单** **🔗** `https://leetcode.cn/problems/two-sum/` **容器：`unordered_map`（属于 map 系列）**

在数组中找两个数使其和等于目标值，返回下标。用 `map` 记录已遍历过的值，把 O(n²) 暴力优化到 O(n)，这道题把 map 的"以值换下标"思想展现得淋漓尽致。

------

### 题5 — 字母异位词分组

**📍 力扣 LeetCode 49 · 难度：中等** **🔗** `https://leetcode.cn/problems/group-anagrams/` **容器：`map` + `vector`**

将一组字符串按字母构成相同（异位词）归为一组。核心思路是对每个字符串排序后作为 `map` 的 key，value 用 `vector<string>` 存储同组结果。同时用到了两个容器，是综合练习的好题。
