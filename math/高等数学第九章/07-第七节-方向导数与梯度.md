# 第七节 方向导数与梯度

## 一、方向导数

### 1. 方向导数的概念

在多元函数微分学中，我们已经学习了偏导数，它描述了函数沿坐标轴方向的变化率。但在实际问题中，我们往往需要知道函数沿任意方向的变化率。例如，在气象学中，我们需要知道温度沿任意方向的变化情况，而不仅仅是沿经线或纬线方向。这就引出了方向导数的概念。

**定义** 设二元函数 $z = f(x, y)$ 在点 $P_0(x_0, y_0)$ 的某邻域 $U(P_0)$ 内有定义，$l$ 为从点 $P_0$ 出发的一条射线，其方向由方向余弦 $(\cos\alpha, \cos\beta)$ 确定。在 $l$ 上任取一点 $P(x_0 + \Delta x, y_0 + \Delta y)$，且 $P \in U(P_0)$，记
$$\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2}$$

若极限
$$\lim_{\rho \to 0^+} \frac{f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0)}{\rho}$$

存在，则称此极限为函数 $f(x, y)$ 在点 $P_0$ 处沿方向 $l$ 的**方向导数**，记作
$$\left.\frac{\partial f}{\partial l}\right|_{(x_0, y_0)} \quad \text{或} \quad f_l(x_0, y_0)$$

注意，$\Delta x = \rho \cos\alpha$，$\Delta y = \rho \cos\beta$，其中 $\cos\alpha, \cos\beta$ 是方向 $l$ 的方向余弦。

**方向导数与偏导数的关系：**
- 当 $l$ 沿 $x$ 轴正方向时，$\cos\alpha = 1, \cos\beta = 0$，方向导数即为 $\frac{\partial f}{\partial x}$。
- 当 $l$ 沿 $y$ 轴正方向时，$\cos\alpha = 0, \cos\beta = 1$，方向导数即为 $\frac{\partial f}{\partial y}$。
- 偏导数是方向导数在坐标轴方向上的特例。

**对三元函数的推广** 类似地，可以定义三元函数 $u = f(x, y, z)$ 在点 $(x_0, y_0, z_0)$ 处沿方向 $l$（方向余弦为 $\cos\alpha, \cos\beta, \cos\gamma$）的方向导数：
$$\left.\frac{\partial f}{\partial l}\right|_{(x_0, y_0, z_0)} = \lim_{\rho \to 0^+} \frac{f(x_0 + \Delta x, y_0 + \Delta y, z_0 + \Delta z) - f(x_0, y_0, z_0)}{\rho}$$
其中 $\rho = \sqrt{(\Delta x)^2 + (\Delta y)^2 + (\Delta z)^2}$，且 $\Delta x = \rho\cos\alpha, \Delta y = \rho\cos\beta, \Delta z = \rho\cos\gamma$。

### 2. 方向导数的存在条件与计算公式

**定理（方向导数的计算公式）** 如果函数 $z = f(x, y)$ 在点 $(x_0, y_0)$ 处可微，则函数在该点沿任意方向 $l$ 的方向导数都存在，且
$$\left.\frac{\partial f}{\partial l}\right|_{(x_0, y_0)} = f_x(x_0, y_0)\cos\alpha + f_y(x_0, y_0)\cos\beta$$

其中 $\cos\alpha, \cos\beta$ 是方向 $l$ 的方向余弦。

**证明** 因为 $f(x, y)$ 在 $(x_0, y_0)$ 可微，所以函数的全增量可表示为
$$f(x_0 + \Delta x, y_0 + \Delta y) - f(x_0, y_0) = f_x(x_0, y_0)\Delta x + f_y(x_0, y_0)\Delta y + o(\rho)$$

将 $\Delta x = \rho\cos\alpha, \Delta y = \rho\cos\beta$ 代入，得
$$\frac{f(x_0 + \rho\cos\alpha, y_0 + \rho\cos\beta) - f(x_0, y_0)}{\rho} = f_x(x_0, y_0)\cos\alpha + f_y(x_0, y_0)\cos\beta + \frac{o(\rho)}{\rho}$$

令 $\rho \to 0^+$，即得结论。

**三元函数的计算公式** 若函数 $u = f(x, y, z)$ 在点 $(x_0, y_0, z_0)$ 可微，则
$$\left.\frac{\partial f}{\partial l}\right|_{(x_0, y_0, z_0)} = f_x(x_0, y_0, z_0)\cos\alpha + f_y(x_0, y_0, z_0)\cos\beta + f_z(x_0, y_0, z_0)\cos\gamma$$

**注意：** 可微是方向导数存在的充分条件，而非必要条件。有些函数在不可微的点处仍可能存在方向导数。

---

**例题1** 求函数 $f(x, y) = x^2 + xy - y^2$ 在点 $(1, 2)$ 处沿方向 $l$ 的方向导数，其中 $l$ 的方向角为 $\alpha = \frac{\pi}{3}, \beta = \frac{\pi}{6}$。

**解** 先求偏导数：
$$f_x(x, y) = 2x + y, \quad f_y(x, y) = x - 2y$$

在点 $(1, 2)$ 处：
$$f_x(1, 2) = 2 \times 1 + 2 = 4, \quad f_y(1, 2) = 1 - 2 \times 2 = -3$$

方向余弦：
$$\cos\alpha = \cos\frac{\pi}{3} = \frac{1}{2}, \quad \cos\beta = \cos\frac{\pi}{6} = \frac{\sqrt{3}}{2}$$

由方向导数计算公式：
$$\left.\frac{\partial f}{\partial l}\right|_{(1, 2)} = 4 \times \frac{1}{2} + (-3) \times \frac{\sqrt{3}}{2} = 2 - \frac{3\sqrt{3}}{2}$$

---

**例题2** 求函数 $f(x, y, z) = xy + yz + zx$ 在点 $(1, 1, 2)$ 处沿从点 $(1, 1, 2)$ 到点 $(3, 4, 6)$ 方向的方向导数。

**解** 先求偏导数：
$$f_x = y + z, \quad f_y = x + z, \quad f_z = y + x$$

在点 $(1, 1, 2)$ 处：
$$f_x = 1 + 2 = 3, \quad f_y = 1 + 2 = 3, \quad f_z = 1 + 1 = 2$$

确定方向向量：从 $(1, 1, 2)$ 到 $(3, 4, 6)$ 的向量为
$$\vec{l} = (2, 3, 4)$$

其模长为：
$$|\vec{l}| = \sqrt{2^2 + 3^2 + 4^2} = \sqrt{4 + 9 + 16} = \sqrt{29}$$

方向余弦：
$$\cos\alpha = \frac{2}{\sqrt{29}}, \quad \cos\beta = \frac{3}{\sqrt{29}}, \quad \cos\gamma = \frac{4}{\sqrt{29}}$$

方向导数：
$$\left.\frac{\partial f}{\partial l}\right|_{(1, 1, 2)} = 3 \times \frac{2}{\sqrt{29}} + 3 \times \frac{3}{\sqrt{29}} + 2 \times \frac{4}{\sqrt{29}} = \frac{6 + 9 + 8}{\sqrt{29}} = \frac{23}{\sqrt{29}}$$

---

**例题3** 讨论函数 $f(x, y) = \sqrt{x^2 + y^2}$ 在点 $(0, 0)$ 处沿任意方向的方向导数。

**解** 函数 $f(x, y) = \sqrt{x^2 + y^2}$ 在 $(0, 0)$ 处不可微（因为偏导数不存在——事实上 $f_x(0, 0)$ 不存在），但我们可以直接按定义求方向导数。

沿方向 $l$（方向余弦 $\cos\alpha, \cos\beta$），有
$$\Delta x = \rho\cos\alpha, \quad \Delta y = \rho\cos\beta$$

方向导数为：
$$\left.\frac{\partial f}{\partial l}\right|_{(0, 0)} = \lim_{\rho \to 0^+} \frac{\sqrt{(\rho\cos\alpha)^2 + (\rho\cos\beta)^2} - 0}{\rho} = \lim_{\rho \to 0^+} \frac{\rho\sqrt{\cos^2\alpha + \cos^2\beta}}{\rho} = 1$$

即函数在 $(0, 0)$ 处沿任意方向的方向导数均为 $1$。这说明方向导数存在并不保证函数可微。

## 二、梯度

### 1. 梯度的概念

方向导数告诉我们函数沿某一方向的变化率，但我们自然要问：在给定点处，函数沿哪个方向变化最快（即方向导数最大）？这个问题的答案引出了梯度的概念。

设函数 $z = f(x, y)$ 在点 $(x, y)$ 处可微，则沿方向 $l$（方向余弦为 $\cos\alpha, \cos\beta$）的方向导数为：
$$\frac{\partial f}{\partial l} = f_x(x, y)\cos\alpha + f_y(x, y)\cos\beta$$

从向量形式看，上式右端是向量 $(f_x, f_y)$ 与方向单位向量 $(\cos\alpha, \cos\beta)$ 的数量积：
$$\frac{\partial f}{\partial l} = (f_x, f_y) \cdot (\cos\alpha, \cos\beta)$$

**定义** 设函数 $z = f(x, y)$ 在平面区域 $D$ 内具有一阶连续偏导数，则对于每一点 $(x, y) \in D$，向量
$$\left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}\right)$$

称为函数 $f(x, y)$ 在点 $(x, y)$ 处的**梯度**，记作
$$\text{grad}\, f(x, y) \quad \text{或} \quad \nabla f(x, y)$$

即
$$\nabla f(x, y) = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}\right)$$

其中符号 $\nabla$ 读作"Nabla"，是一个向量微分算子：
$$\nabla = \left(\frac{\partial}{\partial x}, \frac{\partial}{\partial y}\right)$$

**三元函数的梯度** 对于三元函数 $u = f(x, y, z)$，其梯度定义为
$$\nabla f(x, y, z) = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

### 2. 梯度与方向导数的关系

由方向导数的计算公式可得：
$$\frac{\partial f}{\partial l} = \nabla f \cdot \vec{e}_l = |\nabla f| \cos\theta$$

其中 $\vec{e}_l = (\cos\alpha, \cos\beta)$ 是方向 $l$ 的单位向量，$\theta$ 是 $\nabla f$ 与 $\vec{e}_l$ 的夹角。

由此可得以下重要结论：

1. **梯度方向是函数增长最快的方向。** 当 $\cos\theta = 1$，即 $\vec{e}_l$ 与 $\nabla f$ 同向时，方向导数取最大值 $|\nabla f|$。因此，梯度方向是使函数值增加最快的方向。

2. **负梯度方向是函数减小最快的方向。** 当 $\cos\theta = -1$，即 $\vec{e}_l$ 与 $\nabla f$ 反向时，方向导数取最小值 $-|\nabla f|$。

3. **梯度的模等于方向导数的最大值：**
$$|\nabla f(x, y)| = \sqrt{\left(\frac{\partial f}{\partial x}\right)^2 + \left(\frac{\partial f}{\partial y}\right)^2}$$

### 3. 梯度的几何意义

对于二元函数 $z = f(x, y)$，其梯度 $\nabla f(x_0, y_0)$ 是这样一个向量：方向与过点 $(x_0, y_0)$ 的等值线 $f(x, y) = C$（其中 $C = f(x_0, y_0)$）的法线方向一致，且指向函数值增加的一方；梯度的模就是法线方向上的方向导数。

**关于等值线：** 对于函数 $z = f(x, y)$，曲线 $f(x, y) = C$ 称为等值线（或等高线）。梯度的方向与等值线在该点的法线方向一致。这是因为在等值线上函数值不变，所以沿切线方向的方向导数为 $0$，即
$$\nabla f \cdot \vec{T} = 0$$
其中 $\vec{T}$ 为等值线的切向量，故梯度一定垂直于等值线（即沿法线方向）。

**关于等值面：** 对于三元函数 $u = f(x, y, z)$，曲面 $f(x, y, z) = C$ 称为等值面。梯度 $\nabla f(x_0, y_0, z_0)$ 的方向与过点 $(x_0, y_0, z_0)$ 的等值面的法线方向一致。

#### 3.1 等值线与等值面的常用结论

**等值线 $f(x,y)=C$**

若 $\nabla f(x_0,y_0)\neq \boldsymbol{0}$，则等值线在点 $(x_0,y_0)$ 处的法向量为

$$
\nabla f(x_0,y_0)=(f_x(x_0,y_0),f_y(x_0,y_0)).
$$

因此等值线的切线方程为

$$
f_x(x_0,y_0)(x-x_0)+f_y(x_0,y_0)(y-y_0)=0.
$$

**等值面 $f(x,y,z)=C$**

若 $\nabla f(x_0,y_0,z_0)\neq \boldsymbol{0}$，则等值面在点 $(x_0,y_0,z_0)$ 处的切平面方程为

$$
f_x(M_0)(x-x_0)+f_y(M_0)(y-y_0)+f_z(M_0)(z-z_0)=0.
$$

法线方程为

$$
\frac{x-x_0}{f_x(M_0)}=\frac{y-y_0}{f_y(M_0)}=\frac{z-z_0}{f_z(M_0)}.
$$

这与第六节曲面切平面公式一致，本质原因是等值面 $f(x,y,z)=C$ 可以写成 $F(x,y,z)=f(x,y,z)-C=0$。

**方向导数与等值线的关系**

若方向单位向量 $\boldsymbol{e}$ 与等值线切向量同向，则

$$
D_{\boldsymbol{e}}f=\nabla f\cdot\boldsymbol{e}=0.
$$

若方向单位向量与梯度同向，则方向导数最大；若与梯度反向，则方向导数最小。

### 4. 梯度的运算法则

梯度运算满足以下基本法则（设 $u, v$ 是可微函数，$C$ 为常数）：

1. **线性性：** $\nabla(Cu) = C\nabla u$，$\nabla(u \pm v) = \nabla u \pm \nabla v$
2. **乘积法则：** $\nabla(uv) = u\nabla v + v\nabla u$
3. **商法则：** $\nabla\left(\frac{u}{v}\right) = \frac{v\nabla u - u\nabla v}{v^2}$（$v \neq 0$）
4. **链式法则：** 若 $F(t)$ 可导，则 $\nabla F(u) = F'(u)\nabla u$

这些运算法则可以由偏导数的相应法则直接推导得出。

**例题4** 求函数 $f(x, y) = x^2 + 3y^2$ 在点 $(1, 2)$ 处的梯度，并求方向导数的最大值。

**解** 求偏导数：
$$f_x(x, y) = 2x, \quad f_y(x, y) = 6y$$

在点 $(1, 2)$ 处：
$$f_x(1, 2) = 2, \quad f_y(1, 2) = 12$$

故梯度为：
$$\nabla f(1, 2) = (2, 12)$$

方向导数的最大值为梯度的模：
$$|\nabla f(1, 2)| = \sqrt{2^2 + 12^2} = \sqrt{4 + 144} = \sqrt{148} = 2\sqrt{37}$$

函数沿梯度方向 $(2, 12)$ 增长最快，对应单位方向为 $\left(\frac{1}{\sqrt{37}}, \frac{6}{\sqrt{37}}\right)$，增长率为 $2\sqrt{37}$。

**例题5** 求函数 $f(x, y) = \ln(x^2 + y^2)$ 在点 $(1, 1)$ 处的梯度，并求出该点处沿方向 $l = (1, 2)$ 的方向导数。

**解** 求偏导数：
$$f_x(x, y) = \frac{2x}{x^2 + y^2}, \quad f_y(x, y) = \frac{2y}{x^2 + y^2}$$

在点 $(1, 1)$ 处：
$$f_x(1, 1) = \frac{2}{1 + 1} = 1, \quad f_y(1, 1) = \frac{2}{1 + 1} = 1$$

梯度为：
$$\nabla f(1, 1) = (1, 1)$$

方向 $l = (1, 2)$ 的单位向量：
$$|\vec{l}| = \sqrt{1^2 + 2^2} = \sqrt{5}, \quad \vec{e}_l = \left(\frac{1}{\sqrt{5}}, \frac{2}{\sqrt{5}}\right)$$

方向导数：
$$\left.\frac{\partial f}{\partial l}\right|_{(1, 1)} = \nabla f(1, 1) \cdot \vec{e}_l = 1 \times \frac{1}{\sqrt{5}} + 1 \times \frac{2}{\sqrt{5}} = \frac{3}{\sqrt{5}}$$

**例题6** 已知函数 $f(x, y)$ 在点 $(2, 3)$ 处沿从 $(2, 3)$ 到 $(3, 4)$ 方向的方向导数为 $2\sqrt{2}$，沿从 $(2, 3)$ 到 $(4, 2)$ 方向的方向导数为 $-2$，求函数 $f$ 在点 $(2, 3)$ 处的梯度。

**解** 设梯度 $\nabla f(2, 3) = (a, b)$。

第一个方向：向量 $(3-2, 4-3) = (1, 1)$，单位向量为 $\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$，方向导数为
$$\frac{\partial f}{\partial l_1} = a \cdot \frac{1}{\sqrt{2}} + b \cdot \frac{1}{\sqrt{2}} = \frac{a + b}{\sqrt{2}} = 2\sqrt{2}$$
即
$$a + b = 4$$

第二个方向：向量 $(4-2, 2-3) = (2, -1)$，单位向量为 $\left(\frac{2}{\sqrt{5}}, -\frac{1}{\sqrt{5}}\right)$，方向导数为
$$\frac{\partial f}{\partial l_2} = a \cdot \frac{2}{\sqrt{5}} + b \cdot \left(-\frac{1}{\sqrt{5}}\right) = \frac{2a - b}{\sqrt{5}} = -2$$
即
$$2a - b = -2\sqrt{5}$$

解方程组：
$$\begin{cases}
a + b = 4 \\
2a - b = -2\sqrt{5}
\end{cases}$$

两式相加得：
$$3a = 4 - 2\sqrt{5}, \quad a = \frac{4 - 2\sqrt{5}}{3}$$

代入第一式得：
$$b = 4 - a = 4 - \frac{4 - 2\sqrt{5}}{3} = \frac{12 - 4 + 2\sqrt{5}}{3} = \frac{8 + 2\sqrt{5}}{3}$$

故梯度为：
$$\nabla f(2, 3) = \left(\frac{4 - 2\sqrt{5}}{3}, \frac{8 + 2\sqrt{5}}{3}\right)$$

**例题7** 求函数 $f(x, y, z) = e^{xy}\sin z$ 在点 $(0, 1, \frac{\pi}{2})$ 处的梯度。

**解** 求偏导数：
$$f_x = ye^{xy}\sin z, \quad f_y = xe^{xy}\sin z, \quad f_z = e^{xy}\cos z$$

在点 $(0, 1, \frac{\pi}{2})$ 处：
$$f_x(0, 1, \frac{\pi}{2}) = 1 \times e^{0} \times \sin\frac{\pi}{2} = 1 \times 1 \times 1 = 1$$
$$f_y(0, 1, \frac{\pi}{2}) = 0 \times e^{0} \times \sin\frac{\pi}{2} = 0$$
$$f_z(0, 1, \frac{\pi}{2}) = e^{0} \times \cos\frac{\pi}{2} = 1 \times 0 = 0$$

故梯度为：
$$\nabla f\left(0, 1, \frac{\pi}{2}\right) = (1, 0, 0)$$

即梯度方向为 $x$ 轴正方向，模为 $1$。

**例题8** 求曲面 $x^2 + y^2 + z^2 = 1$ 在点 $\left(\frac{1}{2}, \frac{1}{2}, \frac{\sqrt{2}}{2}\right)$ 处的切平面方程和法线方程。

**解** 令 $F(x, y, z) = x^2 + y^2 + z^2 - 1$，则曲面是等值面 $F(x, y, z) = 0$。

梯度为：
$$\nabla F = (2x, 2y, 2z)$$

在点 $\left(\frac{1}{2}, \frac{1}{2}, \frac{\sqrt{2}}{2}\right)$ 处：
$$\nabla F\left(\frac{1}{2}, \frac{1}{2}, \frac{\sqrt{2}}{2}\right) = \left(1, 1, \sqrt{2}\right)$$

该梯度即为切平面的法向量。切平面方程为：
$$1 \times \left(x - \frac{1}{2}\right) + 1 \times \left(y - \frac{1}{2}\right) + \sqrt{2} \times \left(z - \frac{\sqrt{2}}{2}\right) = 0$$
即
$$x + y + \sqrt{2}z = 2$$

法线方程为：
$$\frac{x - \frac{1}{2}}{1} = \frac{y - \frac{1}{2}}{1} = \frac{z - \frac{\sqrt{2}}{2}}{\sqrt{2}}$$

**例题9** 设 $f(x, y)$ 可微，且 $f(1, 1) = 3$，$f_x(1, 1) = 2$，$f_y(1, 1) = -1$。求函数 $g(t) = f(t, t^2)$ 在 $t = 1$ 处的导数，并解释其与梯度的关系。

**解** 由复合函数求导法则：
$$g'(t) = f_x(t, t^2) \times 1 + f_y(t, t^2) \times 2t$$

在 $t = 1$ 处，$(t,t^2)=(1,1)$，所以
$$g'(1) = 2 \times 1 + (-1) \times 2 = 0$$

这里曲线 $(t,t^2)$ 在 $t=1$ 处的切向量为 $(1,2)$，而 $g'(1)=\nabla f(1,1)\cdot(1,2)$，表示 $f$ 沿该曲线的瞬时变化率为零。

**例题10** 设 $f(x, y)$ 可微，且 $\nabla f(2, -1) = (3, 4)$，求 $f$ 在点 $(2, -1)$ 处方向导数的最大值、最小值以及方向导数取最大值的方向。

**解** 方向导数的最大值等于梯度的模：
$$\max \frac{\partial f}{\partial l} = |\nabla f(2, -1)| = \sqrt{3^2 + 4^2} = 5$$

方向导数的最小值为其相反数：
$$\min \frac{\partial f}{\partial l} = -|\nabla f(2, -1)| = -5$$

取最大值的方向为梯度方向，即
$$\vec{l}_{\text{max}} = (3, 4)$$
其单位向量为 $\left(\frac{3}{5}, \frac{4}{5}\right)$。

## 三、方向导数与梯度的常见题型

### 1. 已知方向向量求方向导数

若题目给出的方向是普通向量 $\boldsymbol{a}=(a,b)$ 或 $\boldsymbol{a}=(a,b,c)$，必须先化为单位向量：

$$
\boldsymbol{e}=\frac{\boldsymbol{a}}{|\boldsymbol{a}|}.
$$

然后使用

$$
D_{\boldsymbol{e}}f(P_0)=\nabla f(P_0)\cdot \boldsymbol{e}.
$$

> **例题11** 求 $f(x,y)=x^2y+y^3$ 在点 $(1,2)$ 处沿向量 $(3,4)$ 方向的方向导数。
>
> **解** 先求梯度：
>
> $$
> f_x=2xy,\quad f_y=x^2+3y^2.
> $$
>
> $$
> \nabla f(1,2)=(4,13).
> $$
>
> 方向向量 $(3,4)$ 的单位向量为
>
> $$
> \boldsymbol{e}=\left(\frac35,\frac45\right).
> $$
>
> 所以
>
> $$
> D_{\boldsymbol{e}}f(1,2)=(4,13)\cdot\left(\frac35,\frac45\right)
> =\frac{12+52}{5}=\frac{64}{5}.
> $$

### 2. 已知两个方向导数反求梯度

若 $\nabla f(P_0)=(a,b)$，且已知两个不平行方向上的方向导数，则可列方程：

$$
\nabla f(P_0)\cdot \boldsymbol{e}_1=d_1,\qquad
\nabla f(P_0)\cdot \boldsymbol{e}_2=d_2.
$$

解出 $a,b$ 即得梯度。

### 3. 最大方向导数与最陡方向

设 $f$ 在 $P_0$ 可微，且 $\nabla f(P_0)\neq \boldsymbol{0}$。

| 问法 | 答案 |
|:---|:---|
| 最大方向导数 | $|\nabla f(P_0)|$ |
| 最大增长方向 | $\nabla f(P_0)$ 或单位向量 $\dfrac{\nabla f(P_0)}{|\nabla f(P_0)|}$ |
| 最小方向导数 | $-|\nabla f(P_0)|$ |
| 最快下降方向 | $-\nabla f(P_0)$ |
| 方向导数为 $0$ 的方向 | 与 $\nabla f(P_0)$ 垂直的方向 |

当 $\nabla f(P_0)=\boldsymbol{0}$ 时，若 $f$ 可微，则沿任意方向的方向导数都等于 $0$。此时不能说“梯度方向是最快增长方向”，因为零向量没有确定方向。

### 4. 等值线与等值面题型

等值线、等值面问题通常按以下步骤：

1. 写出等值函数 $F$，使等值线或等值面为 $F=C$；
2. 求梯度 $\nabla F$；
3. 在指定点代入，得到法向量；
4. 二维中写切线，三维中写切平面和法线。

> **例题12** 求等值面 $x^2+y^2-z=0$ 在点 $(1,2,5)$ 处的切平面和法线。
>
> **解** 令
>
> $$
> F(x,y,z)=x^2+y^2-z.
> $$
>
> $$
> \nabla F=(2x,2y,-1).
> $$
>
> 在 $(1,2,5)$ 处，
>
> $$
> \nabla F(1,2,5)=(2,4,-1).
> $$
>
> 切平面方程为
>
> $$
> 2(x-1)+4(y-2)-(z-5)=0,
> $$
>
> 即
>
> $$
> 2x+4y-z=5.
> $$
>
> 法线方程为
>
> $$
> \frac{x-1}{2}=\frac{y-2}{4}=\frac{z-5}{-1}.
> $$

### 5. 易错点

1. 方向导数公式中的方向必须是单位方向，不能直接把普通方向向量代入。
2. 方向导数的定义中 $\rho\to0^+$，表示沿射线方向趋近；若研究双向直线方向，要注意正反方向的符号可能相反。
3. 可微可以推出任意方向导数存在，但任意方向导数存在不一定推出可微。
4. 梯度垂直于等值线或等值面，不是沿着等值线或等值面。
5. 方向导数最大值是 $|\nabla f|$，不是梯度向量本身。
6. 若题目问“下降最快方向”，应取负梯度方向。

## 习题 9-7

**1.** 求函数 $f(x, y) = x^2 - y^2$ 在点 $(1, 1)$ 处沿方向 $l$ 的方向导数，其中 $l$ 的方向与 $x$ 轴正方向夹角为 $45^\circ$。

**2.** 求函数 $f(x, y, z) = x^2 + y^2 + z^2$ 在点 $(1, 2, -1)$ 处沿从该点到点 $(2, 3, 1)$ 方向的方向导数。

**3.** 求函数 $f(x, y) = e^{x+y}$ 在点 $(0, 0)$ 处沿方向 $l = (1, 2)$ 的方向导数。

**4.** 求函数 $f(x, y) = x^2y + xy^2$ 在点 $(1, 2)$ 处的梯度。

**5.** 求函数 $f(x, y, z) = \sin(xy) + \cos(yz)$ 在点 $(1, 0, \frac{\pi}{2})$ 处的梯度。

**6.** 已知函数 $f(x, y)$ 在点 $(1, 2)$ 处沿从 $(1, 2)$ 到 $(2, 3)$ 方向的方向导数为 $3\sqrt{2}$，沿从 $(1, 2)$ 到 $(4, 2)$ 方向的方向导数为 $6$，求 $f$ 在点 $(1, 2)$ 处的梯度。

**7.** 证明：函数 $f(x, y) = \sqrt{x^2 + y^2}$ 在点 $(0, 0)$ 处沿任何方向的方向导数存在且等于 $1$，但函数在该点不可微。

**8.** 求曲面 $z = x^2 + y^2$ 在点 $(1, 2, 5)$ 处的切平面方程和法线方程。

**9.** 设函数 $f(x, y)$ 可微，且 $f(1, 2) = 4$，$f_x(1, 2) = 2$，$f_y(1, 2) = 3$，求 $f$ 在点 $(1, 2)$ 处方向导数的最大值和最小值。

**10.** 设 $f(x, y) = x^3 - 3x^2y + 3xy^2 + y^3$，求 $\nabla f(1, -1)$。

### 习题解答

**1. 解** 
$$f_x = 2x, \quad f_y = -2y$$
在 $(1, 1)$ 处：$f_x = 2, f_y = -2$。方向 $l$ 的方向余弦：$\cos45^\circ = \frac{\sqrt{2}}{2}, \cos45^\circ = \frac{\sqrt{2}}{2}$。
$$\frac{\partial f}{\partial l} = 2 \times \frac{\sqrt{2}}{2} + (-2) \times \frac{\sqrt{2}}{2} = 0$$

**2. 解**
$$f_x = 2x, \quad f_y = 2y, \quad f_z = 2z$$
在 $(1, 2, -1)$ 处：$f_x = 2, f_y = 4, f_z = -2$。
方向向量：$(2-1, 3-2, 1-(-1)) = (1, 1, 2)$，模 $\sqrt{1+1+4} = \sqrt{6}$。
方向余弦：$\left(\frac{1}{\sqrt{6}}, \frac{1}{\sqrt{6}}, \frac{2}{\sqrt{6}}\right)$。
$$\frac{\partial f}{\partial l} = 2 \times \frac{1}{\sqrt{6}} + 4 \times \frac{1}{\sqrt{6}} + (-2) \times \frac{2}{\sqrt{6}} = \frac{2 + 4 - 4}{\sqrt{6}} = \frac{2}{\sqrt{6}} = \frac{\sqrt{6}}{3}$$

**3. 解**
$$f_x = e^{x+y}, \quad f_y = e^{x+y}$$
在 $(0, 0)$ 处：$f_x = 1, f_y = 1$。方向 $l = (1, 2)$ 的单位向量：$\left(\frac{1}{\sqrt{5}}, \frac{2}{\sqrt{5}}\right)$。
$$\frac{\partial f}{\partial l} = 1 \times \frac{1}{\sqrt{5}} + 1 \times \frac{2}{\sqrt{5}} = \frac{3}{\sqrt{5}}$$

**4. 解**
$$f_x = 2xy + y^2, \quad f_y = x^2 + 2xy$$
在 $(1, 2)$ 处：$f_x = 2\times1\times2 + 2^2 = 4 + 4 = 8$，$f_y = 1^2 + 2\times1\times2 = 1 + 4 = 5$。
$$\nabla f(1, 2) = (8, 5)$$

**5. 解**
$$f_x = y\cos(xy), \quad f_y = x\cos(xy) - z\sin(yz), \quad f_z = -y\sin(yz)$$
在 $(1, 0, \frac{\pi}{2})$ 处：
$$f_x = 0 \times \cos 0 = 0$$
$$f_y = 1 \times \cos 0 - \frac{\pi}{2} \times \sin 0 = 1 - 0 = 1$$
$$f_z = -0 \times \sin 0 = 0$$
$$\nabla f\left(1, 0, \frac{\pi}{2}\right) = (0, 1, 0)$$

**6. 解** 设 $\nabla f(1, 2) = (a, b)$。

第一个方向：$(2-1, 3-2) = (1, 1)$，单位向量 $\left(\frac{1}{\sqrt{2}}, \frac{1}{\sqrt{2}}\right)$
$$\frac{a + b}{\sqrt{2}} = 3\sqrt{2} \Rightarrow a + b = 6$$

第二个方向：$(4-1, 2-2) = (3, 0)$，单位向量 $(1, 0)$
$$a \times 1 + b \times 0 = 6 \Rightarrow a = 6$$

代入得 $b = 0$，故 $\nabla f(1, 2) = (6, 0)$。

**7. 证明**
沿方向 $l$（方向余弦 $\cos\alpha, \cos\beta$），按定义：
$$\frac{\partial f}{\partial l} = \lim_{\rho \to 0^+} \frac{\sqrt{(\rho\cos\alpha)^2 + (\rho\cos\beta)^2}}{\rho} = \lim_{\rho \to 0^+} \frac{\rho}{\rho} = 1$$
故沿任意方向的方向导数均为 $1$。

但 $f_x(0, 0)$ 不存在（由偏导定义 $\lim_{\Delta x \to 0} \frac{|\Delta x|}{\Delta x}$ 不存在），故函数在 $(0, 0)$ 处不可微。

**8. 解** 令 $F(x, y, z) = x^2 + y^2 - z$，则
$$\nabla F = (2x, 2y, -1)$$
在 $(1, 2, 5)$ 处：$\nabla F(1, 2, 5) = (2, 4, -1)$。

切平面方程：
$$2(x-1) + 4(y-2) + (-1)(z-5) = 0 \Rightarrow 2x + 4y - z = 5$$

法线方程：
$$\frac{x-1}{2} = \frac{y-2}{4} = \frac{z-5}{-1}$$

**9. 解**
方向导数的最大值等于梯度的模：
$$\max \frac{\partial f}{\partial l} = |\nabla f(1, 2)| = \sqrt{2^2 + 3^2} = \sqrt{13}$$

最小值：
$$\min \frac{\partial f}{\partial l} = -\sqrt{13}$$

**10. 解**
$$f_x = 3x^2 - 6xy + 3y^2, \quad f_y = -3x^2 + 6xy + 3y^2$$
在 $(1, -1)$ 处：
$$f_x = 3 \times 1^2 - 6 \times 1 \times (-1) + 3 \times (-1)^2 = 3 + 6 + 3 = 12$$
$$f_y = -3 \times 1^2 + 6 \times 1 \times (-1) + 3 \times (-1)^2 = -3 - 6 + 3 = -6$$
$$\nabla f(1, -1) = (12, -6)$$
